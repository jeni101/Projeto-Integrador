# Release Notes - v0.1.0-dashboard

**Data:** 20 de Maio de 2026  
**Status:** Auditada e Homologada  
**Commit de Origem:** `[release] release-1.md`

## 1. Escopo da Release

Esta release engloba a entrega do MVP da interface do Dashboard Geral de monitoramento da Horta.

* **Tela Principal:**
  * Navbar: Base para atualizações futuras, contendo histórico, crescimento e sensores.
  * Tendencias em Tempo Real (Gráfico): Por enquanto mockadas, mas permite atualizações futuras de desenvolvimento.
  * Cards: Cards que mostram a atual: Temperatura, umidade do ar e umidade solo.
  * Alertas: Caso algum problema aconteça com os equipamentos auxiliares a automação.
  * Tabela de erros: Caso algum alerta seja disparado, ou não disparado.
  * Status Dispositivo: Verificação real de problemas e Idle.
  * Ação Rápida: Caso seja necessário irrigar a distância.
  * Simular Estados do Dispositivo: Caso queira ver mudanças de tela caso não esteja no estado normal.

## 2. Escopo Postergado

Os itens abaixo não estão adicionados nesta release.

* **Informações Reais:**
  * Todas as informações presentes no dashboard são mockadas.

* **Telas Não Principais:**
  * Histórico: Mensagens e informações anteriores.
  * Cescimento: Informações dependentes de histórico.
